## 📋 Deskripsi Program
Program ini adalah aplikasi manajemen workout harian yang memungkinkan pengguna untuk:
- Menambahkan sesi workout (jenis olahraga, durasi, kalori, tanggal)
- Melihat daftar workout yang sudah dilakukan
- Menghitung total kalori yang terbakar
- Menyimpan dan memuat data workout
